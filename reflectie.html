Vakken

Web Apps From Scratch

Het was een goede introductie van de minor. Ook fijn dat we samen moesten werken in een groep zodat we elkaar meteen konden leren kennen. Wel een beetje stressvol omdat we minder dan twee weken de tijd hadden.

CSS To The Rescue

Makkelijk het beste vak van de minor. Er was een mogelijkheid om heel creatief te zijn en los te gaan in CSS. De opdracht was niet alleen leuk, maar de gastdocenten waren ook enorm goed. Elke gastdocent had wel iets unieks om mee te geven naar ons. Al met al, super leuk vak.

Browser Technologies

Dit vak was een beetje stressvol, maar ook waardevol. We moesten een belasting formulier maken en verbeteren. Het enige kritiek puntje wat ik kan geven is dat het onderwerp wat leuker kon. Een erfbelasting aangifte? Ugh... Verder wel goede gastdocenten en ik heb zeker iets geleerd over formulieren!

Hackathon

Leuke sprint met een leuk concept. We hebben als groepje een kaart gemaakt met alle afkomsten van de sprekers van elk jaar. Het was een beetje stressvol omdat we maar een weekje hadden, maar het was zeker een geslaagd project.

Human Centered Design

Een unieke opdracht: we moesten een website maken voor een specifiek persoon met een beperking. In mijn geval moest ik een website maken voor een blinde vrouw. Het was zeker een unieke ervaring en ook een zeldzame kans om echt te gaan testen met een blind persoon. Op deze manier konden we echt met precisie deze website voor haar maken. Ik ben niet enorm blij met mijn eigen resultaat, maar ik vond het wel een leuke opdracht.

API

Bij API moesten we een app maken waarbij je een API gebruikt om de informatie op te halen. Ik heb besloten om een app te maken die gebruik maakt van de Schiphol API. Hiermee kan je zien wat de status is van alle vliegtuigen van of naar Schiphol vliegen. Het was opzich een leuke opdracht en ik heb een redelijk prima website gemaakt. Het enige kritiek wat ik heb op dit vak is dat we meer van dit nodig hebben. Ik heb altijd moeite gehad met JavaScript en dit was eigenlijk de enige kans die ik heb gekregen om daar iets mee te doen. Er wordt veel meer liefde gegeven aan CSS-achtige vakken en er wordt meer gezegd dat we minder JavaScript moeten gebruiken. Dus dit was een goed moment om iets meer aandacht te besteden aan JavaScript
